# Qatar_World_Cup_app
This is an application to make the Qatar World Cup more exciting!

このアプリはカタールW杯での知りたい情報やみんなと情報を共有できるアプリになっている。

## 概要：

カタールW杯に関する情報(試合日程、選手情報、出場国)とログインして他の人とチャットできる機能を有している。\
